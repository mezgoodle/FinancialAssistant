## Commands

1. Run dev server:

```bash
fastapi dev
```

2. Add new dependencies:

```bash
uv add fastapi[standard]
```

3. Remove dependencies:

```bash
uv remove fastapi[standard]
```

4. Update dependencies:

```bash
uv update
```

5. Install dependencies:

```bash
uv sync
```

6. Update dependencies and lock file:

```bash
uv update --all-extras
```
